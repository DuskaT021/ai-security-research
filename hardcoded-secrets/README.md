# Hardcoded secrets

Tracks how API keys, tokens, and connection strings end up in source, env-less frontends, and committed config when speed wins over hygiene. Includes ways to detect them and sanitized examples of real-world shapes.

- [`detection/`](./detection/) — scripts and notes for scanning repos for exposed material
- [`examples/`](./examples/) — sanitized patterns (what *not* to commit)
